# CertiCheck AI

AI-based certificate verification system using OCR + pattern detection.

## Tech Stack
- Node.js (Backend)
- Python (OCR + ML processing)
- HTML/CSS/JS (Frontend)

## Features
- Certificate validation using image analysis
- OCR extraction
- Pattern detection
- Chat-based interface

## How to run
1. npm install
2. node server.js
3. Run Python scripts for OCR/ML modules
